# To-Do List Web Application

## Project Description

This project is a simple **To-Do List Web Application** developed using **HTML, CSS, and JavaScript**. The application allows users to add tasks with a due date, display the tasks in a list, and delete tasks when they are completed.

This project was developed as part of an **internship submission** to demonstrate basic front-end web development skills including DOM manipulation, event handling, and responsive layout design.

---

## Features

* Add new tasks
* Assign a due date to each task
* Display tasks dynamically
* Delete tasks from the list
* Simple and clean user interface
* Organized project structure with separate HTML, CSS, and JavaScript files

---

## Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling and layout of the application
* **JavaScript (ES6)** – Functionality and dynamic interaction

## How to Run the Project

1. Download or clone the project folder.
2. Open the project folder in your system.
3. Open the file **12-todolist.html** in any web browser.
4. Enter a task name and select a due date.
5. Click the **Add** button to add the task.
6. Use the **Delete** button to remove tasks from the list.
